About the Spam Classifier

Project Information

This is a machine learning-based Email/SMS spam classifier that can identify whether a message is spam or not.

How it Works

The system uses a trained machine learning model to classify messages. Here's the process:

  1. Text Preprocessing: When you enter a message, it's first cleaned and preprocessed.
  2. Feature Extraction: The preprocessed text is converted into numerical features using TF-IDF vectorization.
  3. Classification: These features are fed into the machine learning model which predicts if the message is spam or not.

Technologies Used

  • Flask - Web framework
  • NLTK - For natural language processing
  • Scikit-learn - For machine learning algorithms
  • HTML/CSS - For the user interface

Model Performance

The spam classification model was trained on a dataset of labeled spam and non-spam messages. The model achieves high accuracy in identifying spam messages.
